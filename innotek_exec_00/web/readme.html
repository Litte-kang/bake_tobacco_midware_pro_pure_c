一、设置中间件ID
     1、在“中间件ID”一栏中第一个输入框输入中间件所在地的地址。

     2、在第二个输入框中输入10位中间件ID。

     3、填写完毕后点击“中间件ID”一栏中的“提交”按钮即可。

二、添加自控仪ID
     1、在“自控仪ID”一栏输入自控仪编号，然后点击“添加”按钮，即会有下图所示结果。

     2、此时如果不需要再添加自控仪编号，点击下图所示的提交按钮即可。

     3、如果需要添加多个自控仪编号，只需重复1步骤多次 最后执行2步骤即可。

三、删除自控仪编号 （一次只能删除一个自控仪编号）
    在“自控仪ID”一栏输入编号然后点击“删除” 按钮，即完成删除操作 。

四、点击“提取自控仪数据”按钮可将中间件目前所能访问的自控仪的数据全部采集上传。
五、如何鉴别自控仪数据是否访问成功
     如某一自控仪编号连续三次出现下图红框或篮筐所示信息表示此台自控仪无法访问到数据，如果出现“{"midAddress":0000000000","type":0,"address":"00001","data":[80,0,1,140,0,240,9,8]}”类似这样的信息表示访问数据成功。
六、当访问不到自控仪数据时如何解决问题。
     1、首先检查自控仪编号是否正确。

     2、检测自控仪与无线模块是否接触良好。

     3、重启中间件再次试试。

     4、以上三种方法都不行请联系本公司技术人员。
